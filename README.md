# TaskTracker — проект для Практической №21

Учебное desktop-приложение для итогового тестирования проекта.

## Что умеет
- регистрация пользователя
- вход пользователя
- создание задач
- удаление задач
- отметка задачи как выполненной
- сохранение задач в JSON

## Важно
В системе специально оставлены дефекты:
- логические ошибки
- ошибки валидации
- проблемы UX
- нестабильное поведение в отдельных сценариях

## Запуск
python tasktracker.py
